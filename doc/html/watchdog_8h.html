class  Watchdog
 

Detailed Description

provides the interface to the Watchdog module

This provides basic Watchdog service for the mbed. You can configure various timeout intervals that meet your system needs. Additionally, it is possible to identify if the Watchdog was the cause of any system restart, permitting the application code to take appropriate behavior.

Adapted from Simon's Watchdog code from http://mbed.org/forum/mbed/topic/508/
